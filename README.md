# 🥬 Spoilage Shield

Spoilage Shield is a smart food management web application that helps users reduce food waste by tracking food items, monitoring expiry dates, and sending timely reminders before products expire. The application promotes sustainable living by making food inventory management simple and efficient.

---

## ✨ Features

- 🔐 Secure User Authentication
- 🥦 Add and Manage Food Items
- ⏰ Expiry Date Tracking
- 🔔 Expiry Notifications & Alerts
- 📊 Dashboard Overview
- 🔍 Search and Filter Food Items
- 📱 Responsive Design
- ☁️ Cloud Data Storage

---

## 🛠️ Tech Stack

- React + Vite
- Tailwind CSS
- Firebase Authentication
- Cloud Firestore
- JavaScript
